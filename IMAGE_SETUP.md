# Daedric Symbol Setup

## Image Installation

Save the Daedric symbol image as **`daedric-symbol.png`** in the root portfolio directory.

### Where to Place:
```
/CS.Portfolio/
├── daedric-symbol.png  ← Save the image here
├── index.html
├── style.css
└── ... other files
```

### Image Requirements:
- **Filename**: `daedric-symbol.png` (exact match)
- **Background**: Transparent PNG works best
- **Size**: At least 400x500 pixels recommended
- **Format**: PNG or JPG

### What the Code Expects:
The CSS will automatically load and display the image with:
- Smooth fade-in animation (3 seconds)
- Glowing green drop shadow
- Breathing glow effect
- Fog swirling around it
- Perfect integration with Skyrim aesthetic

### Image Already Used:
If you saved it as something else, just rename it to `daedric-symbol.png`

That's it! The portfolio will do the rest. 🎮⚔️
